# 6.1 호출 흐름 상세 — 번호 사이 내용을 채운 전체 시퀀스

`전체 계획.md` 6.1절 표는 상태 전이 간 호출만 나열하고, 그 사이의 내부 처리·
판정·분기는 생략되어 있다. 아래는 그 내부 처리를 같은 시퀀스 안에 끼워 넣어
1번부터 다시 번호를 매긴 것이다. (원본 표 자체는 `전체 계획.md` 6.1절 참고,
성공 경로만 다룬다 — 실패/재시도 분기는 3절 상태표 참고.)

## 정리

빠졌던 내부 처리는 대부분 세 가지 유형이었다.

1. **물리적 대기** — 로봇 이동 완료, LLM 응답 대기, STT 추론 시간 (2, 5, 9, 23, 29, 35번 등)
2. **내부 판정 로직** — 트리거 조건, 삼중 확인, 발산/abort 판정 (11, 13, 14, 21번 등)
   — 3절 상태표에는 있지만 원래 6.1절 표에는 생략되어 있었음
3. **구독 주체 전환** — 12→17번처럼 같은 토픽을 누가 듣는지가 바뀌는 지점
