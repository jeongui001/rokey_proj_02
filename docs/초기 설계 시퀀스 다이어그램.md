# 초기 설계 시퀀스 다이어그램 (drawio 원본 텍스트화)

> 출처: `rokey_proj_02.drawio (1).png`. 이 문서는 그림에 적힌 내용을 **그대로**
> 표로 옮긴 것으로, 초기 기획 당시의 시퀀스를 기록하기 위한 문서다.
>
> **주의 — 실제 구현과 다른 부분이 있다.** 이 그림에는 `PARSING` 상태, "문장을
> llm을 활용하여 분석", `RELEASE` 상태, `TRACK_HAND`(현재는 `APPROACH_HAND`)가
> 등장하지만, 실제 구현은 LLM을 쓰지 않고(동기 키워드 파싱), `PARSING`·`RELEASE`
> 상태 자체가 없다. 현재 상태 머신·실제 구현 기준 내용은 `전체 계획.md` 3절을
> 참고할 것 — 이 문서는 그 이전 시점의 원안이다.
