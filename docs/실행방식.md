## 전체 실행 (터미널 2개 필요)
roboton
ros2 launch task_manager handover_system.launch.py 

## ros2 jazzy (Python 3.12) - dsr_controller2 포크 차이 때문에 local_params_file 필요
ros2 launch task_manager handover_system.launch.py local_params_file:=/home/rokey/rokey_proj_02/dev/local_params.yaml

stt_node(음성 웨이크워드)는 handover_system.launch.py에서 제외되어 있다(현재 안 씀).
필요하면 아래 "노드별 개별 실행" 섹션의 stt_node 명령으로 따로 켤 수 있다.

hardware_enabled 기본값이 true로 바뀌어서, 위 명령을 그대로 실행하면
실제 M0609/RG2에 명령이 전송됩니다. dry_run으로 실행하려면:
ros2 launch task_manager handover_system.launch.py hardware_enabled:=false

## 노드별 개별 실행 (디버깅용)

전체를 다 켜지 않고 특정 패키지만 켜고 싶을 때.

### robot_control
ros2 launch robot_control robot_control.launch.py
ros2 launch robot_control robot_control.launch.py hardware_enabled:=false
ros2 run robot_control robot_control_node
ros2 run robot_control probe_speedl_stream   # speedl_stream 워치독 진단 스크립트, 실행 시 yes 확인 필요

### vision_node
ros2 launch vision_node vision_node.launch.py   # realsense + TF + vision_node + tool_detection_node
ros2 run vision_node vision_node
ros2 run vision_node tool_detection_node
ros2 run vision_node fake_detection_publisher   # 실 카메라 없이 detection 토픽 흉내

pose 모델 keypoint 확인:
ros2 topic echo /vision/tool_track --field kpt0_x
ros2 topic echo /vision/tool_track --field kpt0_y
ros2 topic echo /vision/tool_track --field kpt1_x
ros2 topic echo /vision/tool_track --field kpt1_y

### task_manager
ros2 launch task_manager task_manager.launch.py
ros2 run task_manager task_manager_node

### stt_node
ros2 launch stt_node stt_node.launch.py
ros2 run stt_node stt_node

### operator_gui
ros2 launch operator_gui operator_gui.launch.py
ros2 run operator_gui operator_gui
